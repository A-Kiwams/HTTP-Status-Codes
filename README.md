# HTTP-Status-Codes
Status codes in HTTP

# 1XX - Information responses

100  Continue

101  Switching protocals

102  Processing

# 2xx - Success messages

200 Ok

201 Created

202 Accepted

203 Non-Authoritative Information

204 No Content

205 Reset Content
